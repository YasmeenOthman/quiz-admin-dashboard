## 🚀 Getting Started

```bash
git clone git@github.com:YasmeenOthman/quiz-admin-dashboard.git
cd quiz-admin-dashboard
npm install
npm run dev
```

## 🔑 Features

- Create quizzes
- Add questions with options
- Toggle active/inactive status
- Delete or edit quizzes
